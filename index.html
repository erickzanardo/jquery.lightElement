LightElement example

This is a simple example showing how to use the lightElement plugin, it is used to focus an element by creating opaque blocks surrounding it. Important this plugin "light" one element at time. So before "lighting" another element it's important to unlight the currently before.

Code

CSS

.default-overlay {
	background-color: #000;
	opacity: 0.7;
	z-index: 3;
}

JavaScript

$(".light-element").click(function() {
	var bt = $('<button class="btn">Close</button>');
	bt.click(function() {
		$("#player").unlightElement();
	});
	$("#player").lightElement({topElement: bt});
});

$(document).on("click", ".default-overlay", function() {
	$("#player").unlightElement();
});
